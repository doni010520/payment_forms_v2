Storage de anexos desativado

Carregando…

Como funciona: Quando habilitado, todos os anexos novos serão enviados para a pasta configurada no OneDrive/SharePoint via Microsoft Graph API. Anexos antigos permanecem onde estão (local). Em caso de falha de upload remoto, o arquivo é mantido localmente como fallback.

Pré-requisito: registrar um App Registration no Azure AD com permissão Files.ReadWrite.All (Application) e gerar um client_secret. O Graph Drive ID pode ser obtido em GET /v1.0/drives/{driveId} ou na URL do SharePoint.
